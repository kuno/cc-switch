# OpenWrt B2.3 Artifacts

This folder keeps the final `B2.3` design handoff artifacts in the repo so the planning PR does not depend on temporary files under `/tmp` or `/Volumes/tmp`.

## Included

- `option-b23-light.png`
- `option-b23-dark.png`
- `option-b23-rationale.txt`
- `prototype/index.html`
- `prototype/icons/`

## Usage

- View the static mocks directly from the PNG files.
- Open `prototype/index.html` in a browser to review the clickable design handoff locally.
